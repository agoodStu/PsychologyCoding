# psychCode
这个仓库用于存放一些在研究生期间用python编写的代码。

### FaceRating

用psychopy写的一个面孔评定程序。代码很简陋，是为了上课交作业。

### behaviorData

处理实验中的行为数据。

### eyemoveData

处理实验中的眼动数据。

### crossSpatialCueingTask

跨通道空间线索任务。

### SmoothPursuitTask

平滑追踪任务。包括两种：

- 基于正弦波的平滑追踪（Lissajous curve）
- 基于八个方向的直线追踪任务

眼动部分使用的EyeLink眼动仪。使用Psychopy和pylink连接。
